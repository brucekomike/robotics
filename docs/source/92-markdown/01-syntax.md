# Markdown 基础语法

## 1. 标题与段落

### 标题级别
```markdown
# H1 标题
## H2 标题
### H3 标题
#### H4 标题
```
**渲染效果：**  
# H1 标题
## H2 标题
### H3 标题
#### H4 标题

### 段落分割
用空行分隔段落：
```markdown
这是第一段落...

这是第二段落
```

## 2. 文字样式

| 样式         | 语法                | 示例         |
|--------------|---------------------|--------------|
| **加粗**     | `**文字**`          | **重要**     |
| *斜体*       | `*文字*`            | *强调*       |
| ~~删除线~~   | `~~文字~~`          | ~~过时~~     |
| `行内代码`   | `` `代码` ``         | `printf()`   |
| 上标         | `x^2^`              | x²           |
| 下标         | `H~2~O`             | H₂O          |

## 3. 列表系统

### 无序列表
```markdown
- 项目一
  - 子项目
    - 三级子项目
* 替代符号
```

### 有序列表
```markdown
1. 第一步
2. 第二步
   1. 子步骤
```

### 任务列表
```markdown
- [x] 完成项
- [ ] 待办项
```

## 4. 代码块

### 基础代码块
````markdown
```python
print("Hello World")
```
````

### 带标题的代码块 (MyST)
````markdown
```{code-block} python
:caption: 示例代码
print("语法高亮")
```
````

## 5. 链接与图片

### 行内链接
```markdown
[百度](https://www.baidu.com)
```

### 引用式链接
```markdown
[参考文档][ref]

[ref]: https://example.com
```

### 图片嵌入
```markdown
![替代文字](logo.png){width="200"}
```

## 6. 表格系统

### 基础表格
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
```

### 复杂表格 (MyST)
```{list-table}
:header-rows: 1
:name: 示例表

* - 功能
  - 支持状态
* - 合并单元格
  - ✅
* - 表格标题
  - ✅
```

## 7. 引用块

### 基础引用
```markdown
> 这是引用内容
> 多行引用
```

### 高级引用 (MyST)
```{note}
这是 MyST 的注释框
```

```{warning}
这是警告提示框
```
